# Security Policy

## Supported Versions

This project is actively maintained. The latest release is supported with security updates. Older releases may not receive updates unless critical vulnerabilities are discovered.

| Release | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |
